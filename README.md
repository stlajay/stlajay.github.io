## stlajay.github.io

See over here this [site](https://stlajay.github.io)


### Jekyll installation
`gem install jekyll bundler`
OR See [here](https://jekyllrb.com/docs/installation/)

### Create a website

`jekyll new website`

`bundle install`

`bundle update` 

`cd website`

`bundle exec jekyll serve`

Add "incremental: true" in `_config.yml`

`bundle exec jekyll serve --incremental`
